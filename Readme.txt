BWD Elementor Addons Pro is a plugin for Elementor that offers various widgets and preset designs to enhance the design capabilities of the page builder.

== Description ==

BWD Elementor Addons is a plugin built on the Elementor page builder platform. It includes several widgets that can be used to create custom designs for your website. One of the main features of the plugin is the variety of preset designs available, which can save time and simplify the design process. Overall, the plugin aims to enhance the functionality and design capabilities of the Elementor page builder.
